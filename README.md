REVO
===

REVO is an open platform for building, deploying and running node.js applications. It allows programmers to assemble complex apps in minutes by reusing existing components.

Install REVO
===

```
$ npm install -g revo
```

Run REVO
===
```
$ sudo revo
revo 0.5.0: 
```
